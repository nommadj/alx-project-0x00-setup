# ALX Project 0x00 Setup

This repository contains the setup for a TypeScript-based project in the ALX Software Engineering program.

## Features

- TypeScript
- ESLint
- No /src directory
- App Router not used
- Import alias configured

## Getting Started

Install dependencies:

\`\`\`bash
npm install
\`\`\`

To compile TypeScript:

\`\`\`bash
npx tsc
\`\`\`
